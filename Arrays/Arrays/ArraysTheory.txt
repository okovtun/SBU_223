Array
Статические и Динамические


				type name[SIZE];
type - тип элементов массива. 
	   Элементы массива могут быть любого существующего типа,
	   элементы одного массива могу быть лишь одного типа.
name - имя массива. Для именования массивов используются такие же 
	   идентификаторы, как и для именования переменных.
SIZE - количество элементов массива, может быть задано только 
	   целочисленным константным значением.

				const int n = 5;
				int arr[n];
